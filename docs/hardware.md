# Hardware

Este arquivo organiza os pontos de hardware que devem ser entendidos antes de qualquer ajuste de performance no Dell G15 5530.

## Itens acompanhados

- Modelo exato do notebook.
- Processador detectado, nucleos e threads.
- Memoria RAM total.
- Velocidade declarada e configurada dos modulos de RAM, sem serial.
- GPUs detectadas pelo Windows.
- Dados da GPU NVIDIA obtidos pelo `nvidia-smi`, quando disponivel.

## Principios

- Coletar apenas dados necessarios para diagnostico.
- Nao publicar service tag, serial number, identificadores de rede ou caminhos pessoais.
- Comparar qualquer comportamento de performance com temperatura, energia e estado dos drivers antes de considerar tuning.
